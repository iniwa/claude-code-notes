/config — MCP・拡張
Model Context Protocolサーバーを使ってClaude Codeにツールを追加拡張する設定。
mcpServers両方
使用するMCPサーバーの定義。プロジェクト共有は .mcp.json、個人用は settings.json に記述。 
"mcpServers": {
  "github": {
    "command": "npx",
    "args": ["-y", "@anthropic-ai/mcp-server-github"],
    "env": { "GITHUB_TOKEN": "ghp_..." }
  },
  "postgres": {
    "command": "npx",
    "args": ["-y", "@anthropic-ai/mcp-server-postgres",
             "postgresql://localhost/mydb"]
  }
}
env（settings.json内）両方おすすめ
settings.jsonに環境変数を直接記述できる（v2.1.89以降）。ラッパースクリプトが不要になった。MCPサーバーのトークン等をまとめて管理するのに便利。 
"env": {
  "GITHUB_TOKEN": "ghp_...",
  "NODE_ENV": "development",
  "MY_API_KEY": "..."
}
allowedMcpServers両方
使用を許可するMCPサーバーのホワイトリスト。チームでのセキュリティ管理用。未設定時は全サーバーが対象。
channelsEnabledユーザーTeam/Enterprise
チャンネル機能を有効化。--channels フラグと組み合わせることでツール承認プロンプトをスマホに転送するなどができる。
