/config — 基本動作
モデル・言語・表示など、個人の作業環境に関わる設定。主に ~/.claude/settings.json に記述。
ユーザー ~/.claude/settings.json プロジェクト .claude/settings.json 両方 どちらでも有効
model両方おすすめ
デフォルトで使用するモデルを固定。毎回 /model で切り替える手間が省ける。 
"model": "claude-opus-4-6"
language両方おすすめ
Claudeの応答言語を固定。日本語環境では必ず設定する。 
"language": "japanese"
alwaysThinkingEnabledユーザー
拡張思考（extended thinking）をセッション間で維持。複雑な推論タスクに有効だが遅くなる。Option+T でセッション中にトグルも可。 
"alwaysThinkingEnabled": true
showThinkingSummariesユーザー
v2.1.89以降デフォルト非表示になった思考サマリーを表示に戻す。 
"showThinkingSummaries": true
showTurnDurationユーザー
各応答にかかった時間を表示。モデル切り替えの判断材料になる。 
"showTurnDuration": true
cleanupPeriodDaysユーザー
チャット履歴の保持日数。デフォルト30日。0 で無効化（書き込みなし）、365 で1年保持。 
"cleanupPeriodDays": 90
CLAUDE_AUTOCOMPACT_PCT_OVERRIDE環境変数おすすめ
自動コンパクションが発動するコンテキスト使用率（%）。デフォルト95%。深いコンテキストが必要なタスクなら85%程度に下げると安定する。 
export CLAUDE_AUTOCOMPACT_PCT_OVERRIDE=85
themeユーザー
テーマ設定。"dark" / "light" / "auto"。 
"theme": "dark"
outputStyleユーザー
応答フォーマットのスタイル設定。~/.claude/output-styles/ にカスタムスタイルを追加して用途別（コードレビュー・ドキュメント等）に切り替えられる。
