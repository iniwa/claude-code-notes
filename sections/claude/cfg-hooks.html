/config — フック
ツール実行の前後に任意のシェルコマンドを差し込める仕組み。lint自動実行・通知・セキュリティチェック等に活用。
フックのトリガーイベントは PreToolUse・PostToolUse・Notification・UserPromptSubmit・SessionStart・SessionEnd・Stop・SubagentStop・PreCompact・PostCompact の10種類。
PostToolUseよく使う
ツール実行後に発火。ファイル保存後にlint・format・テストを自動実行するのに最適。 
"hooks": {
  "PostToolUse": [{
    "matcher": "Write(*.py)",
    "hooks": [{
      "type": "command",
      "command": "python -m black $CLAUDE_FILE_PATHS"
    }]
  }]
}
PreToolUse
ツール実行前に発火。入力を検証したり、危険なコマンドをブロックしたりするのに使う。プロンプトインジェクションの検出にも活用可能。
Notification
Claudeが処理を完了してユーザーの入力を待つ時に発火。長時間タスクの完了通知に使える（音・デスクトップ通知・Slack等）。
Stop / SubagentStop
Claudeがタスクを終了したと判断した時に発火。「まだ終わっていない、次のステップを続けて」といった継続指示を自動で送るのに使う。
PostCompact新機能
コンパクション完了後に発火。重要な状態をファイルに保存したり、コンパクション後の追加指示を自動送信するのに使える。
SessionStart / SessionEnd
セッション開始・終了時に発火。環境の初期化やログの記録・クリーンアップに使う。
