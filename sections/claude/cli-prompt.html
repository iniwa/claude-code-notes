CLIフラグ — システムプロンプト
Claudeの動作指示を起動時に渡すフラグ。通常は追記系を使う。
--append-system-prompt "..."推奨
デフォルトのシステムプロンプトに追記。デフォルト動作を維持しながらカスタム指示を加える。ほぼこちらを使う。 
gh pr diff "$1" | claude -p \
  --append-system-prompt "セキュリティエンジニアとしてレビュー" \
  --output-format json
--system-prompt "..."
デフォルトプロンプトを完全置換。デフォルト動作が不要な場合のみ使用。ファイルから読み込む場合は --system-prompt-file。
