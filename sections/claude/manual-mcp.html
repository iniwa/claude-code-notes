MCP セットアップ
Codex / Claude Code に MCP（Model Context Protocol）サーバーを追加する手順書。

Codex の MCP 設定

Codex は MCP サーバーをグローバル設定（~/.codex/config.toml）で管理する。CLI で追加する場合は次の形。 
codex mcp add playwright -- npx -y @playwright/mcp@latest
codex mcp add git -- uvx mcp-server-git
codex mcp add tavily -- npx -y tavily-mcp
実体は TOML に保存される。 
[mcp_servers.serena]
startup_timeout_sec = 15
command = "uvx"
args = ["--from", "git+https://github.com/oraios/serena", "serena", "start-mcp-server", "--project-from-cwd", "--context=codex"]

[mcp_servers.playwright]
command = "npx"
args = ["-y", "@playwright/mcp@latest"]

[mcp_servers.git]
command = "uvx"
args = ["mcp-server-git"]

[mcp_servers.tavily]
command = "npx"
args = ["-y", "tavily-mcp"]
登録確認: 
codex mcp list
Codex セッション開始後、必要に応じて次のように指示する。 
Serena で現在のディレクトリをプロジェクトとして有効化してください

Claude Code の MCP 管理コマンド

claude mcp list              # 登録済み MCP 一覧
claude mcp add <name> ...    # MCP を追加
claude mcp remove <name>     # MCP を削除
スコープ指定:
--scope user : 全プロジェクトで有効（デフォルト）
--scope project : そのプロジェクトのみで有効（.claude/settings.json に保存）

共通グローバル登録セット

Codex と Claude Code の両方に入れておく基本セット。Claude Code 側は --scope user を付けると全プロジェクト共通になる。 
claude mcp add --scope user playwright -- npx -y @playwright/mcp@latest
claude mcp add --scope user git -- uvx mcp-server-git
claude mcp add --scope user tavily -- npx -y tavily-mcp

Tavily API キー

API キーは設定ファイルに直書きせず、OS のユーザー環境変数 TAVILY_API_KEY に保存する。Windows では次のコマンドで永続化できる。 
[Environment]::SetEnvironmentVariable("TAVILY_API_KEY", "tvly-...", "User")
設定後は Codex / Claude Code / ターミナルを開き直す。現在のセッションだけで試す場合は次の形。 
$env:TAVILY_API_KEY = "tvly-..."

1. Serena（コードナビゲーション）

serena必須
コードの構造解析・シンボル検索・安全な編集を行う MCP。LSP ベースで動作。 
claude mcp add --scope user serena -- uvx --from git+https://github.com/oraios/serena serena start-mcp-server --context=claude-code --project-from-cwd
プロジェクトインデックス作成（CLI 上で自然言語で指示）: 
「Serena でこのプロジェクトをインデックスしてください」
Claude が Serena の onboarding ツールを呼び出し、自動でインデックスを作成する。
CLAUDE.md への記載: Use Serena MCP tools for code navigation and editing (find_symbol, find_referencing_symbols, get_symbols_overview, replace_symbol_body, etc.)

2. Tavily（Web 検索）

tavily必須
Web 検索・ページ抽出・クロールを行う MCP。無料枠（月 1,000 クレジット）で利用。
API キー取得: https://app.tavily.com/home 
claude mcp add --scope user tavily -- npx -y tavily-mcp
使えるツール: tavily_search, tavily_extract, tavily_crawl, tavily_research

3. その他の推奨 MCP

playwright推奨
ブラウザ操作（ページ遷移・クリック・スクリーンショット撮影・フォーム入力・要素検査）を Claude Code から実行。Web UI のテストやデバッグに有用。 
claude mcp add --scope user playwright -- npx -y @playwright/mcp@latest
git
リポジトリの履歴・diff・blame を MCP 経由で操作。 
claude mcp add --scope user git -- uvx mcp-server-git
filesystem
特定ディレクトリへのファイル操作権限を明示的に付与。 
claude mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem "/path/to/allowed-directory"

優先度ガイド

必須Serena
コードナビ・編集の基本インフラ
必須Tavily
Web 検索・最新情報取得
推奨Playwright
ブラウザ操作・Web UI テスト

トラブルシューティング

MCP が認識されない
claude mcp list で登録確認。Claude Code を再起動して再試行。
Serena のインデックスが古い
CLI 上で再インデックスを指示: 
「Serena でこのプロジェクトを再インデックスしてください」
Tavily のクレジットが足りない
組み込みの WebFetch ツールを使う（クレジット消費なし）。search_depth: "basic" を使う。
