作業前に見るページ
Codex で意図を作り、Claude Code で実装を回す。最初にここで方針を確認するための作業ダッシュボード。

今日の使い方

Codex に頼む
  • 目的整理
  • 設計判断
  • handoff 作成
  • 差分レビュー
Claude Code に頼む
  • handoff 実装
  • 検証
  • 変更報告
Claude Code が戻すもの
  • changed files
  • summary
  • verification
  • blocked checks
  • design questions

迷ったら

  • 曖昧なら Codex。要求整理と設計判断は Codex で閉じる。
  • 明確で編集量が多いなら Claude Code。Goal / Files / Constraints / Verification がそろってから渡す。
  • 実装後は Codex で設計整合性レビュー。差分を AGENTS.md と照合する。
