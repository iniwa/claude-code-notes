Claude Code の個人用リファレンスサイト
⚠ ご注意

このサイトは いにわ（iniwa） 本人が Claude Code を使って作業するにあたってのメモ書きをまとめたものです。

記載されている設定・手順・ワークフローは個人の環境・用途に合わせたものであり、すべての方にそのまま当てはまるとは限りません。

参考にされる際は、ご自身の環境や要件に合わせて適宜読み替えてください。

このサイトについて

コマンド、フラグ、設定、セットアップ手順、テンプレート集を一つの SPA にまとめた個人用リファレンスです。

  • 作業フロー — プロジェクトセットアップから実装までの手順
  • テンプレート — CLAUDE.md や各種設定ファイルのテンプレート
  • マニュアル — 環境構築・MCP・各種セットアップ手順
  • スラッシュコマンド / CLI フラグ / 設定 — Claude Code のリファレンス
  • ガイド — メモリ、コスト最適化、サブエージェントなどの実践ガイド
