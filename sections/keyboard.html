# メモClaudeへの記憶メモを追加

Shift + Tab で切り替わる3モード

通常モード
デフォルト
ツール使用・ファイル編集のたびに確認プロンプトが出る。慎重に進めたいタスクに。
自動承認モード
auto-accept
編集・ツール実行を自動承認。確認なしで進む。信頼できるタスクや慣れてきたら使いやすい。
プランモード
plan mode
実行前に計画だけを提示。何をするか確認してから実行したい場合に便利。複雑な変更の前に使う。
