テンプレート集
公開用の汎用テンプレート。個人環境に依存する詳細設定は含めない。
いつ使うか
  • plan.md — Codex 側で要求を整理する時。変更種別、設計リスク、Codex か Claude Code かの判断、handoff 化の前提条件をまとめる。
  • AGENTS.md (Codex) — 新規プロジェクトに役割分担と handoff ルールを持ち込む時。
  • CLAUDE.md (Claude Code) — Claude Code に handoff 駆動の作業ルールを与える時。
  • .claudeignore — Claude Code に読ませない範囲を指定する時。
  • .claude/settings.json (project / user) — 権限・フックなどを宣言する時。プロジェクト用とユーザー共通で別ファイル。
