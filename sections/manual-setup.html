プロジェクトセットアップ
新規プロジェクトを Claude Code で作業開始する際のチェックリストと手順書。

1. .gitignore の設定

プロジェクトルートに .gitignore を作成し、以下を含める。 
# ===== Claude Code =====
.claude/

# ===== Serena MCP =====
.serena/

# ===== Python =====
__pycache__/
*.pyc
*.pyo
.venv/
venv/
*.egg-info/

# ===== AI/ML モデルファイル（大容量のため必ず除外） =====
*.bin
*.safetensors
*.gguf
*.pt
*.pth
models/
weights/

# ===== ログ・一時ファイル =====
*.log
*.tmp
*.temp
logs/
tmp/

# ===== 機密情報 =====
.env
.env.*
*.pem
*.key
secrets/

# ===== OS / エディタ =====
Thumbs.db
.DS_Store
*.swp
.vscode/
.idea/
注意: .claude/ にはプロジェクト固有の Claude の記憶・タスク・許可設定が入る。.serena/ には Serena が生成したコードインデックスが入る。どちらも個人環境依存のため、リポジトリには含めない。

2. CLAUDE.md の配置

CLAUDE.md は Claude Code がプロジェクトを開いたとき自動的に読み込む指示書。プロジェクトルートに置く。
CLAUDE.md（最小構成）
# CLAUDE.md

> 詳細（日本語）: CLAUDE_ja.md

## Communication
- User writes in Japanese; respond in English unless instructed otherwise.

## Environment
| Item | Detail |
|------|--------|
| OS   | Windows 11 |
| GPU  | RTX 5060 Ti — 16GB VRAM (CUDA sm_89) |
| RAM  | 64GB |

## Development Policy
- Purpose: [このプロジェクトの目的を記入]
- Language: Python (default for ML tasks)
- Env: `uv` or `venv`; avoid conda unless required

## Tooling
- Use Serena MCP tools for code navigation and editing

## Code Style
- Minimal, readable code over abstraction-heavy patterns
- No CI/CD or packaging unless explicitly requested
CLAUDE.md は AI が読む英語の簡潔版、CLAUDE_ja.md は人間が読む日本語の詳細版という運用が推奨。テンプレートは「CLAUDE.md Docker版」「CLAUDE.md Windows版」を参照。

3. .claudeignore の配置

Claude Code がコンテキストに読み込むファイルの除外設定。.gitignore と独立して動作する。特に ML プロジェクトでモデルファイルを誤って読み込まないために重要。テンプレートは「テンプレート集」を参照。

4. MCP の登録・確認

プロジェクトで使用する MCP サーバーが正しく登録・動作しているか確認する。
  • /config → mcpServers に必要な MCP が登録されているか確認
  • 各 MCP のツールが Claude Code から呼び出せるかテスト
  • 新規 MCP の追加が必要な場合は「MCP セットアップ」を参照

5. Serena のプロジェクトインデックス作成

Serena を使用するプロジェクトでは、Claude Code の CLI 上で自然言語でインデックス作成を指示する。 
「Serena でこのプロジェクトをインデックスしてください」
Claude が Serena の onboarding ツールを呼び出し、自動でプロジェクトインデックスを作成する。

6. 新規プロジェクト チェックリスト

.gitignore 作成（.claude/, .serena/ を含む）
CLAUDE.md をプロジェクトルートに配置
CLAUDE_ja.md を同ディレクトリに配置（任意だが推奨）
.claudeignore を配置（ML プロジェクトはモデルファイルの除外を確認）
MCP サーバーの登録・動作を確認
Serena のプロジェクトインデックスを作成（CLI で自然言語指示）
Python 環境は uv or venv でセットアップ（conda は原則使わない）
GPU 使用有無を決め、CUDA バージョンと PyTorch ビルドが一致しているか確認

7. ディレクトリ構成の目安

project-name/

├── CLAUDE.md

└── src/
