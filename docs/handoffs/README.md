# Handoffs

Codex writes Claude Code implementation handoffs in this directory.

Use the filename format:

```text
YYYY-MM-DD-<short-task>.md
```

Each handoff should be scoped to one implementation task and should include goal, background, files to inspect, files to edit, constraints, non-goals, verification, and expected report format.

Do not store secrets, credentials, or local runtime data in handoff files.
